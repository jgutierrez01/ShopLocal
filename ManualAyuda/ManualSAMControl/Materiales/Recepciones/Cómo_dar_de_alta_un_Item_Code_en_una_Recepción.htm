Dar de Alta un Item Code en Recepciones

Dentro de la pantalla de alta/edición del número único:
1 - Clic en el botón  del campo Item Code.
2 - Seleccionar si se desea dar de alta un nuevo Item Code, o bien si se desea importar uno ya existente en otro proyecto.
3 - Introducir los campos requeridos.
   3.1 - En caso de ser alta introducir: item code, descripción y clasificación.
   3.2 - En caso de ser importación seleccionar el proyecto y el item code a importar.
4 - Clic en Guardar

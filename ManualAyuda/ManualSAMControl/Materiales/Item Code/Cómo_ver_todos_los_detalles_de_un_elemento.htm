Consulta de Materiales por Item Code

1 - Ingresar al módulo de Materiales en el menú principal.
 2 - Seleccionar Listado de Materiales por Item Code
3 - Seleccionar proyecto.
4 - Clic en Mostrar
5 - Aquellos Item Codes que cuenten con equivalencias tendrán la marca   
   5.1 - Al clic, el listado se expanderá mostrando los item codes equivalentes al registro en particular.

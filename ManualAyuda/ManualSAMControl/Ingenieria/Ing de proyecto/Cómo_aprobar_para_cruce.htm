Aprobar para Cruce

1 - Ir al listado de Ingeniería de Proyecto en el módulo de Ingeniería.
2 - Seleccionar un proyecto
3 - Clic en Mostrar
4 - Para aprobar un set de spools para cruce:
    4.1 - Realizar los filtros necesarios para tener visibles sólo los spools que se deseen aprobar
    4.2 - Clic en Aprobado para Cruce
    4.3 - Todos los spools que se encuentren visibles dentro del grid serán aprobados.
5 - Para aprobar un spool en específico, hacer clic sobre el ícono  del elemento
    5.1 - Seleccionar la opción Aprobado para cruce
    5.2 - Clic en Guardar

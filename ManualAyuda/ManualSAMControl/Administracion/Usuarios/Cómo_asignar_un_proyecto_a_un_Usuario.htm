Asignar un Proyecto a un Usuario
  1. Ingresar al módulo de Administración en el menú principal.
  2. Seleccionar Usuarios en el submenú ó parte Inferior de Administración.
  3. Seleccionar el icono  del Usuario a editar.
  4. Seleccionar los proyectos a los que se desea el usuario tenga permisos.
  5. Guardar, para almacenar los datos.

Editar los Item Codes de un proyecto

Agregar un nuevo Item Code

1 - En la sección de LINKS DE NAVEGACIÓN haga click en Item Codes .  
2 - Clic en Agregar.
    2.1 - El sistema lo redireccionará al detalle del Item Code
3 - Introducir todos los campos requeridos
4 - Clic en Guardar

Editar un Item Code

1 - En la sección de LINKS DE NAVEGACIÓN haga click en Item Codes .  
2 - Seleccionar el ícono  del Item Code a Editar.
    2.1 - El sistema lo redireccionará al detalle del Item Code
3 - Introducir todos los campos requeridos
4 - Clic en Guardar

Eliminar un Item Code

1 - En la sección de LINKS DE NAVEGACIÓN haga click en Item Codes .  
2 - Seleccionar el ícono  del Item Code a eliminar.
